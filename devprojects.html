2022 Habit Tracker

Video Demo GitHub Repository

This project is built using the Flask web framework and was built as my final project for Harvard's cs50. I look forward to refining this concept and building out the app further.

This web application allows you to track habits on daily basis. You can track as many habits as you like as well as add and remove them as time goes on and your needs change. Don't worry though! If you remove a habit, you can still look back on your tracked history of that habit. I have preloaded all dates for the 2022 calendar year for this web application.

Shield Platform Encryption

I planned, presented, and executed a project roadmap to migrate from the third-party encryption scheme, CipherCloud, to the Salesforce native platform solution, Shield Platform Encryption. This required careful coordination with the IT department to ensure integrated processes and data uploads to the Salesforce instance continued to function seamlessly during the migration.

Once completed, connected service complexity decreased, therefore improving data upload, extraction, and manipulation processes. This migration also improved overall performance for the entire userbase.

Finally, I maintained documentation on the process to be presented to examiners.
